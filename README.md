# Bachelor Thesis Ecology

## Тема дипломної роботи
**Оцінка екологічного стану водних ресурсів Житомирської області на прикладі річки Гнилоп'ять**

Спеціальність: **101 Екологія**

## Структура проєкту

```text
bachelor-thesis-ecology/
├── thesis/
│   ├── sections/
│   ├── figures/
│   └── tables/
├── data/
│   ├── raw/
│   └── processed/
├── sources/
│   ├── literature/
│   ├── legislation/
│   └── maps/
├── analysis/
│   ├── scripts/
│   └── results/
├── presentation/
├── notes/
└── docs/
```

## Призначення папок

- `thesis/` — основний текст дипломної роботи.
- `thesis/sections/` — розділи пояснювальної записки.
- `thesis/figures/` — рисунки, карти, схеми.
- `thesis/tables/` — таблиці для роботи.
- `data/raw/` — первинні дані без змін.
- `data/processed/` — оброблені дані.
- `sources/literature/` — наукові джерела та бібліографія.
- `sources/legislation/` — закони, нормативи, методики.
- `sources/maps/` — карти, геодані, просторові матеріали.
- `analysis/scripts/` — скрипти для розрахунків або обробки даних.
- `analysis/results/` — результати аналізу.
- `presentation/` — матеріали для захисту.
- `notes/` — робочі нотатки.
- `docs/` — додаткові документи.

## Основні файли

- `thesis/title-page.md` — титульна сторінка.
- `thesis/abstract.md` — анотація.
- `thesis/contents.md` — зміст.
- `thesis/sections/01-introduction.md` — вступ.
- `thesis/sections/02-literature-review.md` — огляд літератури.
- `thesis/sections/03-object-and-methods.md` — об'єкт і методи дослідження.
- `thesis/sections/04-results.md` — результати дослідження.
- `thesis/sections/05-environmental-assessment.md` — оцінка екологічного стану.
- `thesis/sections/06-recommendations.md` — рекомендації.
- `thesis/sections/07-conclusions.md` — висновки.
- `sources/bibliography.md` — список використаних джерел.
